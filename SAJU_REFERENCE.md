# 명리처방전 — 사주명리학 핵심 레퍼런스 가이드

> 이 문서는 "명리처방전" 프로덕트의 사주 분석 품질을 담보하기 위한 정통 명리학 레퍼런스를 정리한 것입니다.
> AI 프롬프트 설계, 사주 계산 로직, 해석 체계 구축 시 반드시 참고해야 합니다.
---
## 0. 해석 체계 선언 (명리처방전 이론 기준)

명리처방전은 **자평명리(子平命理)** 체계를 기본 해석 프레임으로 사용합니다.

- 해석의 중심은 **일간(日干)** 입니다.
- 모든 관계 분석은 **십신(十神) 체계**를 기반으로 전개됩니다.
- 격국·용신 판단은 자평명리의 원칙을 따릅니다.

고법 명리 요소(납음, 신살, 일부 삼명법 구조)는  
보조 참고 지표로만 사용하며,  
용신·격국 판단보다 우선하지 않습니다.

## 2. 3대 용신론 체계 (프로덕트 핵심 이론)

사주 해석의 가장 중요한 축인 **용신(用神)** 이론은 3대 고전에서 각기 다른 관점을 제시합니다.
명리처방전의 AI 분석은 이 3가지 용신론을 **종합적으로** 활용해야 합니다.

| 용신론 | 출처 | 핵심 원리 | 주로 판단하는 영역 |
|--------|------|-----------|-------------------|
| **억부용신** | 적천수 | 일간의 강약을 판단, 강하면 억제·약하면 부조 | 재물운, 가족관계, 개인 사생활 |
| **격국용신** | 자평진전 | 월지와 일간의 십성관계로 격국 수립 | 사회활동, 직장생활, 대외적 성패 |
| **조후용신** | 궁통보감 | 계절·기후 관점에서 사주의 온냉조습 균형 | 기본 체질, 건강, 환경 적응력 |

> **프로덕트 적용 원칙:** 하나의 사주에 대해 억부·격국·조후를 모두 분석한 뒤, 사용자의 고민 영역에 가장 부합하는 용신론을 우선 적용하여 답변합니다.

**오행 상생·상극 관계:**

```
          상생 (→ 낳아줌)
    木 → 火 → 土 → 金 → 水 → 木
   (나무가 불을 낳고, 불이 재(흙)를 낳고, 흙이 금속을 낳고, 금속이 물을 맺고, 물이 나무를 키운다)

          상극 (→ 억제함)
    木 → 土 → 水 → 火 → 金 → 木
   (나무가 흙을 뚫고, 흙이 물을 막고, 물이 불을 끄고, 불이 금속을 녹이고, 금속이 나무를 벤다)
```

| 관계 | 설명 | 상세 |
|------|------|------|
| **상생(相生)** | 낳아주는 관계, 돕는 관계 | 木→火, 火→土, 土→金, 金→水, 水→木 |
| **상극(相剋)** | 억제하는 관계, 제어하는 관계 | 木→土, 土→水, 水→火, 火→金, 金→木 |
| **생아(生我)** | 나를 낳아주는 것 = 인성 | 나를 돕고 보호하는 기운 |
| **아생(我生)** | 내가 낳는 것 = 식상 | 내 재능과 표현의 기운 |
| **극아(剋我)** | 나를 억제하는 것 = 관성 | 나를 통제하고 규율하는 기운 |
| **아극(我剋)** | 내가 억제하는 것 = 재성 | 내가 다스리고 얻는 기운 |
| **동아(同我)** | 나와 같은 것 = 비겁 | 경쟁하고 돕는 동류의 기운 |

### 4-2-2. 신강(身强)·신약(身弱) 판단 기준

일간의 강약은 사주 해석의 출발점입니다:

**신강 조건 (아래 중 2개 이상 충족):**
1. 일간이 월령(월지)에서 왕지(건록·제왕·장생)에 해당
2. 천간에 비겁(비견·겁재)이 있음
3. 천간에 인성(정인·편인)이 있음
4. 지지에 일간과 같은 오행이 많음 (통근이 강함)

**신약 조건 (아래 중 2개 이상 충족):**
1. 일간이 월령에서 사절지(사·묘·절)에 해당
2. 관성(편관·정관)이 강하게 일간을 극함
3. 식상(식신·상관)이 많아 일간의 기운을 빼앗김
4. 재성(편재·정재)이 많아 일간이 극하느라 기운 소모

**신강·신약에 따른 용신 방향:**

| 상태 | 필요한 것 (용신 방향) | 해로운 것 (기신 방향) |
|------|---------------------|---------------------|
| **신강** | 식상·재성·관성 (기운을 빼주는 것) | 비겁·인성 (더 강하게 만드는 것) |
| **신약** | 비겁·인성 (기운을 도와주는 것) | 관성·식상·재성 (더 약하게 만드는 것) |

### 4-12. 대운 계산법

```
1. 성별 + 년간 음양으로 순행/역행 결정:
   - 남자 + 양간(甲丙戊庚壬) = 순행 (→ 다가올 절기)
   - 남자 + 음간(乙丁己辛癸) = 역행 (← 지난 절기)
   - 여자는 반대

2. 생일~해당 절기까지의 일수 계산

3. 일수 ÷ 3 = 대운 시작 나이 (나머지 0,1 버림 / 2 올림)

4. 대운 간지는 월주에서 순행/역행으로 하나씩 이동:
   - 순행: 월주 → 다음 간지 → 다음 간지 ...
   - 역행: 월주 → 이전 간지 → 이전 간지 ...

5. 각 대운은 10년 단위
```

## 5. 궁합 분석 프레임워크

### 5-1. 연인 궁합 분석 포인트

1. **일주 간 관계:** 일간·일지의 합충 관계 확인
2. **오행 상생·보완:** 서로의 부족한 오행을 보완하는가
3. **용신 관계:** 상대가 내 용신 역할을 하는가
4. **궁위(宮位) 분석:** 일지(배우자궁)의 상태
5. **대운 흐름:** 두 사람의 대운이 시기적으로 맞는가

### 5-2. 동료 궁합 분석 포인트

1. **오행 구성 균형:** 팀으로서 오행이 고르게 분포되는가
2. **십신 보완:** 서로 다른 십신 특성으로 역할 분담 가능한가
3. **격국 호환:** 두 사주의 격국이 상충하지 않는가
4. **충·형 관계:** 업무적 마찰 요소 존재 여부

### 5-3. 친구 궁합 분석 포인트

1. **일간 관계:** 비견·겁재 관계 (동질성) vs 식상·재성 관계 (다양성)
2. **지지 합 관계:** 삼합·육합이 있으면 자연스러운 친밀감
3. **성정(性情) 매칭:** 오행 구성이 만드는 성격 궁합
4. **충·원진 관계:** 장기적 갈등 요소

## 8. 정통 해석 템플릿 (AI 프롬프트 참조용)

> AI가 사주를 해석할 때 아래 템플릿 구조를 따라야 정통 명리학 수준의 결과를 낼 수 있습니다.

### 8-1. 일간 해석 템플릿

```
[일간 해석 공식]
"당신은 {일간} 일간입니다. {일간의 자연물 비유}에 비유되는 당신은 {핵심 성정 2~3가지}한 성격입니다.

{월령(월지)}월에 태어나 {계절}의 기운을 받았으므로, {계절이 일간에 미치는 영향}.
일간이 월지에서 {12운성}에 해당하여, {12운성의 의미에 따른 해석}.

사주 전체의 오행 분포를 보면, {강한 오행}이 강하고 {약한 오행}이 부족합니다.
이는 {강한 오행의 성질}이 도드라지는 반면, {약한 오행의 성질}에 대한 보완이 필요함을 뜻합니다."
```

**예시 (甲일간, 午월생):**
> "당신은 甲木 일간입니다. 큰 나무에 비유되는 당신은 곧고 정직하며 리더십이 강한 성격입니다.
> 
> 午月(5월, 여름)에 태어나 화(火)기운이 가장 강한 계절을 만났습니다. 나무가 한여름 뜨거운 햇볕 아래 있는 형상이라, 재능(식신-火)은 넘치지만 목(木) 자체의 에너지가 소모되기 쉽습니다. 일간이 월지에서 '사(死)'에 해당하여, 열정은 크나 체력적 에너지 관리가 중요합니다.
> 
> 사주 전체의 오행을 보면 화(火)와 토(土)가 강하고 수(水)가 부족합니다. 이는 표현력과 실행력은 뛰어나지만, 내면의 깊이와 인내심 보완이 필요함을 뜻합니다."

### 8-2. 대운·세운 해석 템플릿

```
[대운 해석 공식]
"현재 {대운 천간}{대운 지지} 대운 ({시작나이}세~{종료나이}세)을 지나고 있습니다.

대운 천간 {대운 천간}은 일간에게 {십신}에 해당합니다.
{십신의 의미}의 기운이 10년간 당신을 감싸고 있으므로, 이 시기는 {해석}.

대운 지지 {대운 지지}은 원국의 {관계있는 글자}와 {합/충/형 관계}를 이루고 있어,
{합충형의 구체적 영향 해석}."

[세운 해석 공식]  
"올해 {세운 천간}{세운 지지}년은, 일간에게 {십신}이 들어오는 해입니다.
{십신이 세운으로 올 때의 의미}.

특히 세운 지지 {세운 지지}이 원국의 {관계있는 글자}와 {관계}를 이루므로,
{상반기/하반기 구분하여 구체적 시기 언급}."
```

### 8-3. 궁합 해석 템플릿

```
[궁합 해석 공식]
"A({A이름})의 일간 {A일간}과 B({B이름})의 일간 {B일간}은 
{오행 관계: 상생/상극/비화} 관계입니다.
{관계의 의미 해석}.

일지(배우자궁)를 보면, A의 일지 {A일지}와 B의 일지 {B일지}가
{합/충/형/해/무관계}를 이루고 있어, {일지 관계 해석}.

오행 보완성을 보면, A에게 부족한 {A약한 오행}을 
B의 {B가 가진 해당 오행}이 채워주고 있어 {보완 해석}.

다만, A의 {특정 글자}와 B의 {특정 글자}가 {충/형} 관계이므로
{갈등 포인트 구체적 설명}. 이 부분은 {구체적 조언}으로 보완할 수 있습니다."
```

### 8-4. 고민 처방 해석 템플릿

```
[고민 처방 공식]
"'{사용자 고민}' — 이 고민을 명리학 관점에서 살펴보겠습니다.

[원인 분석]
당신의 사주에서 {고민과 관련된 십신/오행}이 {상태 설명}.
{구체적 사주 글자 근거 제시}.
현재 대운 {대운}이 {십신}으로 작용하여 {대운이 고민에 미치는 영향}.
올해 세운 {세운}이 원국의 {글자}와 {관계}를 이루므로 {세운 영향}.

[처방]
시기: {유리한 시기 특정} (근거: {세운/월운 근거})
행동: {구체적 실천 조언} (근거: {용신/오행 보완 논리})
주의: {피해야 할 시기/행동} (근거: {기신/충 관계})

[전망]
{내년 세운}으로 넘어가면 {변화 예측}. 
지금은 {현재 조언 한마디}."
```

### 8-5. 해석 품질 체크리스트

AI 응답 생성 후 아래를 반드시 확인:

| # | 체크 항목 | 기준 |
|---|----------|------|
| 1 | **근거 명시** | 모든 해석에 사주 글자·관계가 근거로 제시되었는가? |
| 2 | **바넘 효과 배제** | "누구에게나 맞는 말"이 포함되어 있지 않은가? |
| 3 | **시기 특정** | "조만간", "곧" 같은 모호한 표현 대신 구체적 년도·월이 명시되었는가? |
| 4 | **교차 검증** | 하나의 해석이 최소 2가지 이상 근거(십신+12운성, 신살+지지관계 등)로 뒷받침되는가? |
| 5 | **균형** | 좋은 점만 나열하거나 나쁜 점만 나열하지 않았는가? |
| 6 | **실천 가능성** | 처방/조언이 구체적이고 실제 행동으로 옮길 수 있는가? |
| 7 | **용어 병기** | 한자 전문 용어 사용 시 쉬운 해석이 함께 제공되었는가? |
| 8 | **일간 중심** | 모든 해석이 일간을 기준으로 전개되었는가? |
| 9 | **궁위 활용** | 년주·월주·일주·시주의 궁위 의미가 적절히 반영되었는가? |
| 10 | **개인화** | 같은 일간이라도 월령·지지관계·신살에 따라 다른 해석이 나오는가? |
