# 명리처방전 — POC 기획서

## 1. 프로덕트 개요

**프로덕트명:** 명리처방전
**한줄 설명:** AI 기반 사주 명리 분석 및 궁합 서비스 — 당신의 고민에 명리학이 처방을 내립니다
**POC 목표:** 핵심 가설 검증 — "고민을 입력하면 사주 기반 맞춤 답변이 사용자에게 가치가 있는가?"

---

## 2. POC 범위

### 포함 (Must Have)

- **사주 정보 입력:** 이름, 성별, 생년월일, 태어난 시간, 양력/음력 선택
- **사주팔자 자동 계산:** manseryeok-js 라이브러리 연동 (정확한 절기력 기반)
- **대운·세운 분석:** 현재 대운(10년 주기)과 올해 세운 계산 및 해석 — 고민 답변의 시간적 근거로 활용
- **고민 입력 → 고민 우선 답변:** 사용자의 고민을 최상단에 배치하여 답변 (핵심 차별점)
- **개인 사주 분석:** 성격, 적성, 재물운, 건강운, 대인관계 종합 분석
- **연인 궁합:** 두 사람의 사주를 입력하여 연인 궁합 분석 (1종만)
- **동양적 + 모던 UI:** 신비로운 분위기의 싱글 페이지

### 제외 (POC 이후)

- 프로필 저장/관리 (매번 입력)
- 동료·친구 궁합 (연인만)
- 사용자 인증/로그인
- 분석 결과 히스토리
- SNS 공유

## 3. 사용자 플로우

```
[홈] → 탭 선택: "내 사주 보기" | "연인 궁합"
         ↓                        ↓
   사주 정보 입력              나 + 상대 사주 입력
         ↓                        ↓
   고민 입력 (선택)            분석 요청
         ↓                        ↓
   분석 결과 표시              궁합 결과 표시
   (고민 답변 최상단)          (궁합 점수 + 해설)
```

### 상세 플로우

1. **홈:** "내 사주 보기" / "연인 궁합" 탭 전환
2. **개인 분석:**
   - 이름, 성별, 생년월일시, 양력/음력 입력
   - "요즘 가장 고민되는 점이 있나요?" 텍스트 입력 (선택)
   - → 결과: 고민 처방전 → 사주 원국표 → 십신 프로파일 → 6대 운세 → 대운·세운 타임라인 → 맞춤 처방
3. **연인 궁합:**
   - 나의 사주 정보 입력 + 상대 사주 정보 입력
   - → 결과: 종합 스코어(5대 영역) → 끌리는 이유 → 갈등 주의보 → 시기별 타임라인 → 관계 처방전 → 결혼운·시점

## 5. 데이터 모델 (런타임 전용, 저장 없음)

```typescript
// 사주 입력 정보
interface SajuInput {
  name: string;
  gender: 'male' | 'female';
  birthDate: string;        // YYYY-MM-DD
  birthTime: string;        // HH:mm 또는 '모름'
  calendarType: 'solar' | 'lunar';
  isLeapMonth?: boolean;    // 음력 윤달 여부
}

// 만세력 라이브러리 계산 결과
interface FourPillars {
  year: { stem: string; branch: string };
  month: { stem: string; branch: string };
  day: { stem: string; branch: string };
  hour: { stem: string; branch: string };
}

// 개인 분석 요청
interface AnalysisRequest {
  pillars: FourPillars;
  input: SajuInput;
  currentDaeun: string;     // 현재 대운 (천간·지지)
  currentSeun: string;      // 올해 세운 (천간·지지)
  daeunStartAge: number;    // 대운 시작 나이
  concern?: string;         // 고민 (선택)
}

// 궁합 분석 요청
interface CompatibilityRequest {
  person1: { input: SajuInput; pillars: FourPillars };
  person2: { input: SajuInput; pillars: FourPillars };
}
```

## 7. AI 프롬프트 설계

### 7-0. 핵심 원칙: "소름끼치게 잘 맞는" 분석을 위한 설계 철학

> 사용자가 "이거 어떻게 알았지?"라고 느끼는 순간이 이 프로덕트의 생명이다.
> 그 느낌은 **바넘 효과(누구에게나 맞는 말)**가 아니라, **진짜 사주 구조에서 도출된 구체적 해석**에서 온다.

**절대 하지 말 것 (바넘 효과 금지 원칙):**
- ❌ "당신은 때로 외로움을 느끼기도 합니다" — 누구에게나 해당
- ❌ "재물운이 좋은 편입니다" — 근거 없는 일반론
- ❌ "노력하면 좋은 결과가 있을 것입니다" — 사주와 무관한 상식

**반드시 할 것 (구체성·개인화 원칙):**
- ✅ "일간 甲木이 월지 午火 위에 앉아 식신이 왕성합니다. 머릿속 아이디어가 끊이지 않는 타입이지만, 생각만 많고 실행이 느린 것이 고민일 수 있습니다" — 사주 구조에서 직접 도출
- ✅ "올해 세운 乙巳가 일간과 겁재 관계라 경쟁이나 예상치 못한 지출이 생기기 쉬운 해입니다. 특히 5~6월(巳月)에 집중될 수 있으니 이 시기 큰 소비는 자제하세요" — 시기까지 특정
- ✅ "배우자궁(일지)에 桃花살이 있어 매력이 넘치는 배우자를 만날 사주입니다. 다만 편관이 혼잡하여 인연이 여럿 겹칠 수 있으니, 한 사람에게 집중하는 것이 결혼으로 가는 지름길입니다" — 구체적 행동 조언

### 7-1. 정확도를 높이는 데이터 설계

**만세력 라이브러리에서 AI에 전달할 데이터 (최대한 풍부하게):**

```typescript
interface SajuAnalysisData {
  // 기본 사주 원국
  fourPillars: FourPillars;          // 년월일시 천간·지지
  fourPillarsHanja: FourPillars;     // 한자 표기
  
  // 오행 분석
  fiveElements: {
    wood: number; fire: number; earth: number;
    metal: number; water: number;
  };
  dominantElement: string;            // 가장 강한 오행
  weakestElement: string;             // 가장 약한 오행
  
  // 십신 관계 (일간 기준)
  tenGods: {
    yearStem: string; yearBranch: string;
    monthStem: string; monthBranch: string;
    dayStem: string; dayBranch: string;
    hourStem: string; hourBranch: string;

  // 지지 관계
  branchRelations: {
    samhap: string[];    // 삼합 관계

  // 지장간

  };

    stem: string; branch: string;

  dayMasterStrength: 'strong' | 'weak' | 'neutral';

### 7-2. 개인 사주 분석 프롬프트

**시스템 프롬프트 핵심 지시:**

```
당신은 30년 경력의 정통 사주명리학 전문가입니다.
적천수의 억부론, 자평진전의 격국론, 궁통보감의 조후론을 모두 통달했습니다.

[분석 원칙]
1. 모든 해석은 반드시 사주 원국의 구체적 글자·관계에서 근거를 밝혀라
   - "甲일간이 子월에 태어나 정인격이며..." 식으로 근거를 먼저 제시
   - 근거 없는 일반론, 누구에게나 맞는 말은 절대 금지

2. 십신·지지 관계·12운성·신살을 종합하여 교차 검증하라
   - 하나의 근거만으로 단정하지 말고 최소 2~3가지 근거가 일치할 때 강하게 진술
   - 근거가 상충할 때는 "~한 면이 있으나 ~한 면도 공존합니다"로 균형 잡기

3. 시기 언급 시 반드시 구체적으로 특정하라
   - ❌ "조만간 좋은 일이 있을 것입니다"
   - ✅ "2026년 丙午 세운이 식신생재를 이루므로, 특히 하반기(7~9월)에 
        새로운 수입원이 열릴 가능성이 높습니다"

4. 사용자의 고민을 사주 구조와 직접 연결하라
   - 고민: "이직을 고민 중이에요"
   - ✅ "현재 대운 庚金이 편관으로 작용하여 조직 내 압박감이 클 시기입니다.
        게다가 올해 세운과 월주가 충을 이루고 있어 직장 변동의 기운이 강합니다.
        다만 내년 세운이 정인이므로, 올해 하반기에 준비하고 내년 상반기에 
        이직하면 좋은 조직을 만날 확률이 높습니다"

5. "처방"은 사주 구조에서 논리적으로 도출하라
   - 용신이 水인 사람에게: "검은색 계열 옷, 북쪽 방향, 겨울 시즌 활동 강화"
   - 단순 미신이 아닌 오행 보완 논리로 설명

6. 톤: 전문적이되 따뜻하게. 딱딱한 학술 용어를 쓰되 반드시 쉬운 해석을 병기
   - "편관(偏官)이 강합니다 — 쉽게 말해 카리스마와 추진력이 넘치지만, 
     주변에서 '너무 세다'는 소리를 들을 수 있습니다"
```

### 7-3. 연인 궁합 프롬프트

**시스템 프롬프트 핵심 지시:**

```
두 사람의 사주를 비교 분석하는 궁합 전문가입니다.

[궁합 분석 원칙]
1. 반드시 두 사주의 구체적 글자 관계를 근거로 제시하라
   - "A의 일지 午와 B의 일지 子가 자오충으로..." 식의 구체적 근거
   - 점수는 감이 아닌, 합·충·형·해·오행보완·십신관계의 가중 합산

2. 궁합 점수 산출 기준 (투명하게 보여주기):
   - 일주 간 관계: 합(+20) / 생(+10) / 극(-10) / 충(-20)
   - 오행 보완도: 서로의 용신을 채워주면 +15
   - 지지 합 관계: 육합(+15) / 삼합(+10) / 방합(+5)
   - 지지 충·형: 충(-15) / 형(-10) / 해(-5)
   - 배우자궁 상태: 길신(+10) / 도화(+5) / 공망(-10)
   - 대운 시기 조화: 동시에 좋은 운(+10) / 엇갈림(-5)
   → 기본 50점에서 가감하여 0~100 범위로 정규화

3. "성적 궁합"은 도화살·함지살·일지 관계를 근거로 하되, 품위 있게 표현

4. 시기별 분석에서 두 사람의 세운을 겹쳐 볼 때:
   - 한쪽이 충운(冲運)이면 관계에 파동이 올 수 있다고 경고
   - 양쪽 모두 합운(合運)이면 결혼·동거 등 관계 진전 시기로 제안

5. 결혼 시점은 다음을 종합:
   - 각자의 관성(남: 정재·편재 / 여: 정관·편관)이 세운·대운에서 들어오는 시기
   - 배우자궁(일지)이 합을 이루는 세운
   - 홍염살·천희성 등 결혼 관련 신살이 작용하는 시기
```

### 7-4. 신뢰도를 높이는 UX 장치

프롬프트뿐 아니라 **결과를 보여주는 방식**도 신뢰감에 큰 영향을 미칩니다:

| UX 장치 | 설명 | 효과 |
|---------|------|------|
| **근거 표시** | 모든 해석 옆에 "근거: 일간甲 + 월지午 = 식신" 식의 태그 표시 | "아무말 대잔치"가 아님을 증명 |
| **사주 원국표** | 한자 사주표를 시각적으로 보여주기 (천간·지지·오행 색상 매핑) | 전문 만세력 느낌, 신뢰감 상승 |
| **오행 차트** | 목·화·토·금·수 비율을 레이더/바 차트로 시각화 | 한눈에 균형/불균형 파악 |
| **타임라인 UI** | 대운·세운 흐름을 타임라인 그래프로 표현 | "운의 흐름"이 시각적으로 와닿음 |
| **궁합 레이더** | 5대 영역별 점수를 레이더 차트로 표현 | 어디가 강하고 약한지 직관적 |
| **핵심 한줄** | 각 섹션 시작에 "당신은 ○○한 사람입니다" 식의 직관적 한줄 | 첫 문장에서 "맞는다!" 느낌 유도 |
| **경고 배지** | 충·형 관계, 공망 등 주의 요소는 빨간 배지로 강조 | 위험 신호를 놓치지 않게 |

**응답 구조:**

**A. 고민 처방전 (고민 입력 시에만, 최상단)**
```
[나의 고민에 대한 명리 처방]
- 고민의 명리학적 원인 분석 (어떤 오행/십신의 작용인지)
- 현재 대운·세운 흐름에서 본 해석
- 구체적 처방: 유리한 시기, 피해야 할 시기, 실천 조언
```

**B. 사주 원국 분석 (기본 체질표)**
```
[사주 원국 — 나의 명리 카드]
- 사주팔자 표 (년·월·일·시 천간/지지 + 한자)
- 일간 해설: "당신은 ○○일간, 한마디로 ○○한 사람"
- 오행 분포 차트: 목·화·토·금·수 각 비율 + 강/약 판정
- 용신·희신·기신 판정: "당신에게 가장 필요한 기운은 ○○"
```

**C. 십신 성격 프로파일**
```
